## 30-Second Quickstart

```bash
# Option 1: Use the CLI scaffolder
npx create-settlegrid-tool --template urlscan

# Option 2: Clone and run
git clone https://github.com/settlegrid/settlegrid-urlscan.git
cd settlegrid-urlscan
npm install
cp .env.example .env   # Add your API keys
npm run dev
```

## Methods

| Method | Description | Cost |
|--------|-------------|------|
| `submit_scan(url)` | Submit a URL for scanning | 2¢ |
| `get_result(uuid)` | Get scan result by UUID | 1¢ |
| `search_scans(query)` | Search existing scans | 1¢ |
| `get_screenshot(uuid)` | Get screenshot URL from scan | 1¢ |
